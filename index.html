NLP

Research Assistant + Class Project

During Spring 2018 I worked as a research assistant for Professor David Chiang, on a project to classify St. Augutine's sermons in Latin. I used the CLTK framework and the CRFSuite library to clean the data and start creating custom models.

During Fall 2018 I took Professor Chiang's NLP class. My final project was a document classifier that attempted to figure out given source text, the news source it came from. We got it up to around 70% accuracy over 16 classes, using the NLTK and SKLearn libraries for Python. Source Code.
